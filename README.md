# Install

```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
Add the following line to your dependencies:
```groovy
dependencies {
	implementation 'com.github.kirvigen:DelegateAdapter:1.0'
}
```
